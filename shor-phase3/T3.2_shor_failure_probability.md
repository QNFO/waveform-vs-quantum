# T3.2: Probability of Shor's Algorithm Failure — Exact Bounds

## 1. Setup

Let $N = \prod_{i=1}^{k} p_i^{e_i}$ be the prime factorization of $N$, with each $p_i$ odd.

By the Chinese Remainder Theorem:

$$
(\mathbb{Z}/N\mathbb{Z})^\times \cong \prod_{i=1}^{k} (\mathbb{Z}/p_i^{e_i}\mathbb{Z})^\times
$$

For odd prime powers, $(\mathbb{Z}/p_i^{e_i}\mathbb{Z})^\times$ is cyclic of order

$$
\phi(p_i^{e_i}) = p_i^{e_i-1}(p_i - 1)
$$

Choose $a \in (\mathbb{Z}/N\mathbb{Z})^\times$ uniformly at random. Define:

- $d_i = \operatorname{ord}_{p_i^{e_i}}(a \bmod p_i^{e_i})$ — the local order at the $i$-th prime power
- $r = \operatorname{ord}_N(a) = \operatorname{lcm}(d_1, d_2, \ldots, d_k)$ — the global order

Let $v_2(x)$ denote the exponent of $2$ in $x$ (2-adic valuation). Then:

$$
v_2(r) = \max_{1 \leq i \leq k} v_2(d_i)
$$

---

## 2. Success and Failure Conditions

Shor's factoring algorithm succeeds when:

> **$r$ is even** AND **$\gcd(a^{r/2} - 1, N)$ yields a non-trivial factor**

This fails in exactly two cases:

### Failure Mode 1: $r$ is odd ($v_2(r) = 0$)

This occurs iff **every** $d_i$ is odd, i.e., $v_2(d_i) = 0$ for all $i$.

### Failure Mode 2: Symmetric Half-Power Failure

$r$ is even ($v_2(r) = t \geq 1$) but for **every** $i$:

$$
a^{r/2} \equiv -1 \pmod{p_i^{e_i}}
$$

By CRT, this is equivalent to $a^{r/2} \equiv -1 \pmod{N}$, in which case $\gcd(a^{r/2} \pm 1, N) \in \{1, N\}$ yields only trivial factors.

This symmetric failure occurs precisely when **all** $d_i$ have the **same** 2-adic valuation $t \geq 1$ AND for each $i$, $a^{d_i/2} \equiv -1 \pmod{p_i^{e_i}}$.

---

## 3. Distribution of 2-Adic Valuations in Cyclic Groups

Let $G$ be a cyclic group of order $2^s \cdot m$ where $m$ is odd. Write $s = v_2(|G|)$.

For a uniformly random $x \in G$:

| $t = v_2(\operatorname{ord}(x))$ | $\Pr[v_2(\operatorname{ord}(x)) = t]$ | Interpretation |
|:---:|:---:|:---|
| $t = 0$ | $2^{-s}$ | Odd order (subgroup of odd $m$) |
| $0 < t < s$ | $2^{t-s-1}$ | Intermediate valuation |
| $t = s$ | $1/2$ | Full 2-adic valuation |

**Verification:**

$$
2^{-s} + \sum_{t=1}^{s-1} 2^{t-s-1} + \frac{1}{2}
= 2^{-s} + \frac{2^{s-1} - 1}{2^s} + \frac{1}{2}
= 2^{-s} + \frac{1}{2} - 2^{-s} + \frac{1}{2} = 1 \quad \checkmark
$$

**Key Lemma (Half-Power Property):** In a cyclic group $C_{2^s}$ for $s \geq 1$, for any $x \in C_{2^s}$ with $\operatorname{ord}(x) = 2^t$ ($t \geq 1$):

$$
x^{2^{t-1}} = -1 \quad \text{(the unique element of order 2)}
$$

*Proof:*
Let $g$ generate $C_{2^s}$, so $x = g^{m}$ for some $m$. Then $\operatorname{ord}(x) = 2^s/\gcd(m, 2^s) = 2^t$.
Writing $m = 2^{v_2(m)} \cdot m_{\text{odd}}$, we have $t = s - v_2(m)$. Then:

$$
x^{2^{t-1}} = g^{m \cdot 2^{t-1}} = g^{m_{\text{odd}} \cdot 2^{v_2(m)} \cdot 2^{s - v_2(m) - 1}} = g^{m_{\text{odd}} \cdot 2^{s-1}} = (g^{2^{s-1}})^{m_{\text{odd}}} = (-1)^{m_{\text{odd}}} = -1
$$

since $m_{\text{odd}}$ is odd. $\square$

Therefore, in each cyclic factor $(\mathbb{Z}/p_i^{e_i}\mathbb{Z})^\times$, **whenever $d_i$ is even** ($v_2(d_i) \geq 1$), we automatically have $a^{d_i/2} \equiv -1 \pmod{p_i^{e_i}}$.

## 4. RSA Semiprime Case: $N = p \cdot q$

Let $s_p = v_2(p-1)$ and $s_q = v_2(q-1)$.

### 4.1 Failure Mode 1 — Both local orders odd

$$
\Pr[\text{odd } r] = \Pr[v_2(d_p) = 0 \land v_2(d_q) = 0] = \frac{1}{2^{s_p}} \cdot \frac{1}{2^{s_q}} = \frac{1}{2^{s_p + s_q}}
$$

### 4.2 Failure Mode 2 — Same valuation, both equal

This occurs when $v_2(d_p) = v_2(d_q) = t \geq 1$ for some $t$.

For a given $t$ (with $1 \leq t \leq \min(s_p, s_q)$):

$$
\Pr[v_2(d_p) = t \land v_2(d_q) = t] = \Pr[v_2(d_p) = t] \cdot \Pr[v_2(d_q) = t]
$$

Using the distribution above:

$$
\Pr[\text{symmetric failure}] = \Pr[v_2(d_p) = v_2(d_q) = s_p = s_q] \cdot 1
+ \sum_{t=1}^{\min(s_p, s_q)-1} \Pr[v_2(d_p) = t] \cdot \Pr[v_2(d_q) = t]
+ \text{(edge case when both at max)}
$$

The bound (used in practice) simplifies the analysis:

### 4.3 Universal Bound for Semiprimes

Shor (1994) proved:

$$
\boxed{\Pr[\text{success}] \geq \frac{1}{2}}
$$

This is a **universal lower bound** for any RSA modulus $N = p \cdot q$ where $\gcd(p-1, q-1)$ can be anything. The bound is tight when $s_p = s_q$ and $\gcd(p-1, q-1)$ contains a large power of 2, in which case the symmetric failure probability approaches $1/2$.

**Worst case:** $p \equiv q \equiv 3 \pmod{4}$ ($s_p = s_q = 1$). Then:

- $\Pr[\text{odd } r] = 1/2 \cdot 1/2 = 1/4$
- $\Pr[\text{symmetric}] = \Pr[v_2(d_p) = v_2(d_q) = 1] = 1/2 \cdot 1/2 = 1/4$
- $\Pr[\text{failure}] = 1/4 + 1/4 = 1/2$
- $\Pr[\text{success}] = 1/2$ ✓

**Best case:** Strong primes ($p = 2p' + 1$, $q = 2q' + 1$ with $p', q'$ odd, so $s_p = s_q = 1$). Same as worst case — 1/2 bound.

When $s_p \neq s_q$ (e.g., $p \equiv 1 \pmod{4}$, $q \equiv 3 \pmod{4}$):

- $s_p \geq 2, s_q = 1$
- $\Pr[\text{odd } r] = 2^{-s_p} \cdot 1/2 = 2^{-s_p-1} \leq 1/8$
- Symmetric failure is impossible (since $v_2(d_p)$ and $v_2(d_q)$ range over different sets)
- $\Pr[\text{success}] \geq 7/8$

## 5. General Case: $k$ Distinct Odd Prime Factors

Let $N = \prod_{i=1}^{k} p_i^{e_i}$ where each $p_i$ is odd. Define $s_i = v_2(\phi(p_i^{e_i})) = v_2(p_i - 1)$.

### 5.1 Probability of Odd Order

$r$ is odd iff **all** $d_i$ are odd:

$$
\Pr[r \text{ odd}] = \prod_{i=1}^{k} \frac{1}{2^{s_i}} = 2^{-\sum_{i} s_i}
$$

Since $s_i \geq 1$ for each odd prime $p_i$:

$$
\boxed{\Pr[r \text{ odd}] \leq \frac{1}{2^k}}
$$

### 5.2 Probability of Symmetric Failure

This requires all $v_2(d_i)$ to be equal to some $t \geq 1$. The probability is:

$$
\Pr[\text{symmetric}] = \sum_{t=1}^{\infty} \prod_{i=1}^{k} \Pr[v_2(d_i) = t]
$$

For each $i$, the probability mass on valuation $t$ is at most:

$$
\max_{1 \leq t \leq s_i} \Pr[v_2(d_i) = t] = \frac{1}{2}
$$

So a crude union-bound gives:

$$
\Pr[\text{symmetric}] \leq \prod_{i=1}^{k} \frac{1}{2} = \frac{1}{2^k}
$$

### 5.3 Total Failure Bound

Combining both modes:

$$
\boxed{\Pr[\text{failure}] \leq \Pr[r \text{ odd}] + \Pr[\text{symmetric}] \leq \frac{1}{2^k} + \frac{1}{2^k} = \frac{1}{2^{k-1}}}
$$

Equivalently:

$$
\boxed{\Pr[\text{success}] \geq 1 - \frac{1}{2^{k-1}}}
$$

### 5.4 Table for $k = 1, 2, 3, 4, 5$

| $k$ (distinct odd prime factors) | $\Pr[\text{failure}] \leq$ | $\Pr[\text{success}] \geq$ |
|:---:|:---:|:---:|
| 1 | 1 (trivial — $N$ is a prime power) | 0 (order-finding does not yield factors) |
| 2 | $1/2$ | $1/2$ |
| 3 | $1/4$ | $3/4$ |
| 4 | $1/8$ | $7/8$ |
| 5 | $1/16$ | $15/16$ |
| $k$ | $1/2^{k-1}$ | $1 - 1/2^{k-1}$ |

**Observation:** As $k$ grows, the success probability rapidly approaches 1, making the algorithm extremely reliable for highly composite numbers.

---

## 6. Edge Cases

### 6.1 Prime Powers: $N = p^e$

$(\mathbb{Z}/p^e\mathbb{Z})^\times$ is cyclic for odd $p$, but Shor's order-finding with a single factor cannot directly produce a factor (the group is not a product). The standard approach would need to explicitly detect perfect powers first (via $N^{1/e}$ checks), which is classical $O(\log^3 N)$ by binary search.

### 6.2 Even Numbers: $N$ divisible by 2

If $2 \mid N$, factor out powers of 2 classically. For $N = 2^e \cdot \prod p_i^{e_i}$, the odd part $N'$ is handed to Shor's algorithm. This is a trivial preprocessing step.

### 6.3 Carmichael Numbers

Carmichael numbers $N$ satisfy $a^{N-1} \equiv 1 \pmod{N}$ for all $a$ with $\gcd(a, N) = 1$, but this does **not** hinder Shor's algorithm. The order $r = \operatorname{ord}_N(a)$ divides the Carmichael function $\lambda(N)$, not $\phi(N)$. Shor's algorithm computes $r$ through QFT period finding regardless of whether $N$ is a Carmichael number. In fact, Carmichael numbers are **easier** for Shor because $\lambda(N)$ is typically smaller, reducing the period length.

However, Carmichael numbers typically have $k \geq 3$ prime factors, which means (per §5.4) the success probability is **higher** ($\geq 3/4$ for $k=3$).

### 6.4 $a$ not coprime to $N$

If $\gcd(a, N) > 1$, we immediately have a non-trivial factor. This is rare for large $N$ ($\Pr[\gcd(a, N) > 1] \approx 1/p$ for RSA semiprime), but when it happens, we get an instant success without running the quantum circuit.

---

## 7. Expected Number of Trials

Let $p_{\text{succ}}$ be the per-trial success probability. We perform independent trials until success.

**Expected trials:** $\mathbb{E}[T] = 1/p_{\text{succ}}$

**Trials for confidence $1 - \varepsilon$:** The probability of $m$ consecutive failures is $(1 - p_{\text{succ}})^m$. Setting this $\leq \varepsilon$:

$$
m \geq \left\lceil \frac{\ln \varepsilon}{\ln(1 - p_{\text{succ}})} \right\rceil
$$

| Success prob $p_{\text{succ}}$ | $\mathbb{E}[T]$ | Trials for 99.9% confidence ($\varepsilon = 10^{-3}$) |
|:---:|:---:|:---:|
| $1/2$ ($k = 2$) | 2 | 10 |
| $3/4$ ($k = 3$) | 1.33 | 5 |
| $7/8$ ($k = 4$) | 1.14 | 3 |
| $15/16$ ($k = 5$) | 1.07 | 2 |

**For RSA-2048** ($k = 2$): Expected 2 trials, or 10 trials for 99.9% confidence. This is a constant overhead.

## 8. Python Implementation

```python
from math import gcd
from functools import reduce
from typing import List, Tuple

def v2(x: int) -> int:
    """2-adic valuation: largest e such that 2^e divides x."""
    if x == 0:
        raise ValueError("v2(0) is undefined")
    e = 0
    while x % 2 == 0:
        x //= 2
        e += 1
    return e

def shor_success_probability(
    factors: List[Tuple[int, int]]
) -> float:
    """
    Compute the exact success probability of Shor's algorithm
    per trial, given the prime factorization of N.

    Args:
        factors: List of (p_i, e_i) tuples for N = ∏ p_i^e_i.
                 Note: p_i must be odd primes.

    Returns:
        Exact probability that a randomly chosen a yields
        a non-trivial factor.

    Derivation:
        For each odd prime power p_i^e_i, the local group is
        cyclic of order φ(p_i^e_i) = p_i^{e_i-1}·(p_i - 1).
        Let s_i = v2(p_i - 1).

        For uniformly random a, the tuple of local 2-adic valuations
        (v2(d_1), ..., v2(d_k)) is independently distributed.

        P(odd r)    = ∏ 2^{-s_i}
        P(symmetric) = Σ_{t=1}^{max s_i} ∏ Pr[v2(d_i) == t]
                     where Pr[v2(d_i) == t] =
                         2^{-s_i}          if t = 0
                         2^{t-s_i-1}       if 0 < t < s_i
                         1/2               if t = s_i

        P(success) = 1 - P(odd r) - P(symmetric)
    """
    odd_primes = [(p, e) for p, e in factors if p > 2]

    if len(odd_primes) == 0:
        # N is a power of 2; factoring is trivial
        return 1.0

    if len(odd_primes) == 1:
        # N = p^e: single cyclic group; order finding alone
        # does not yield factors without additional structure.
        # In practice, one first checks perfect powers,
        # then factors the odd prime power by other means.
        return 0.0  # Shor's standard method is inapplicable

    k = len(odd_primes)
    s_values = [v2(p - 1) for p, _ in odd_primes]
    max_s = max(s_values)

    # P(odd r): all d_i are odd
    prob_odd = 1.0
    for s in s_values:
        prob_odd *= 1.0 / (1 << s)  # 2^{-s}

    # P(symmetric): all valuations equal to same t >= 1
    prob_symmetric = 0.0
    for t in range(1, max_s + 1):
        prob_t = 1.0
        for s in s_values:
            if t < s:
                prob_t *= 2.0 ** (t - s - 1)
            elif t == s:
                prob_t *= 0.5
            else:
                prob_t = 0.0  # t exceeds s_i: impossible
                break
        prob_symmetric += prob_t

    return 1.0 - prob_odd - prob_symmetric


def expected_trials(p_success: float, epsilon: float = 1e-6):
    """Expected number of trials and trials for confidence 1-ε."""
    expected = 1.0 / p_success
    trials_for_confidence = 1
    if p_success < 1.0:
        import math
        trials_for_confidence = math.ceil(
            math.log(epsilon) / math.log(1 - p_success)
        )
    return expected, trials_for_confidence


# ─── Examples ───────────────────────────────────────────

if __name__ == "__main__":
    # RSA-2048 style semiprime: p, q both ≈ 2^1024
    # Both primes ≡ 3 mod 4 (worst case: s_p = s_q = 1)
    print("=== RSA Semiprime (k=2, s_p=s_q=1, worst case) ===")
    p_succ = shor_success_probability([(3, 1), (7, 1)])
    print(f"P(success) = {p_succ:.6f}")
    exp_trials, conf_trials = expected_trials(p_succ)
    print(f"Expected trials: {exp_trials:.2f}")
    print(f"Trials for 99.9% confidence: {conf_trials}")

    # Mixed 2-adic valuations: p ≡ 1 mod 4, q ≡ 3 mod 4
    print("\n=== Mixed valuations (s_p=3, s_q=1) ===")
    # p=17 (17-1=16, s=4), q=7 (7-1=6, s=1) — wait, v2(6)=1
    # p=17: v2(16)=4, q=7: v2(6)=1
    p_succ = shor_success_probability([(17, 1), (7, 1)])
    print(f"P(success) = {p_succ:.6f}")

    # Three primes: k=3
    print("\n=== Three primes (k=3, all s=1: worst case) ===")
    p_succ = shor_success_probability([(3, 1), (7, 1), (11, 1)])
    print(f"P(success) = {p_succ:.6f}")

    # Five primes: k=5
    print("\n=== Five primes (k=5) ===")
    p_succ = shor_success_probability(
        [(3, 1), (5, 1), (7, 1), (11, 1), (13, 1)]
    )
    print(f"P(success) = {p_succ:.6f}")

    # RSA-2048 typical: large primes
    print("\n=== RSA-2048 typical (two 1024-bit primes) ===")
    # s_p, s_q are typically 1..10 for random large primes
    # Probability of both s=1 is about (1/2)^2 = 1/4
    # But the bound P(success) >= 1/2 is universal
    print("Universal bound: P(success) ≥ 1/2 ✓")
```

## 9. Key Takeaway

| Result | Statement |
|--------|-----------|
| **RSA semiprime** | $\Pr[\text{success}] \geq 1/2$ per trial — universal bound |
| **$k$ distinct primes** | $\Pr[\text{failure}] \leq 2^{-(k-1)}$, $\Pr[\text{success}] \geq 1 - 2^{-(k-1)}$ |
| **Expected trials** | $\mathbb{E}[T] = 1/p_{\text{succ}} \leq 2$ for RSA, $\leq 4/3$ for $k=3$ |
| **Confidence scaling** | $O(\ln(1/\varepsilon))$ trials for confidence $1-\varepsilon$ |
| **Worst case** | $k=2$ with $s_p = s_q = 1$ ($p \equiv q \equiv 3 \bmod 4$) |

The constant per-trial overhead ($\leq 2$ expected trials) means Shor's algorithm's practical viability depends **entirely** on the per-trial runtime (quantum circuit depth × qubit count), not on the number of trials.
